INCIDENT RESPONSE TOOL

Local-First Incident Journal

장애 대응 중 메모를 빠르게 남기면, 타임라인을 자동 정리해 포스트모템 초안을 즉시 생성합니다. 목표는 "기록 누락 최소화 + 문서화 시간 단축"입니다.

빠른 사용법 (1분)

  1. ID/제목/심각도를 입력합니다.
  2. 이벤트를 줄 단위로 입력합니다: kind|author|text
  3. 포스트모템 생성을 누르면 하단에 초안이 생성됩니다.
kind 예시 detected impact_update mitigation root_cause_note recovery action_item

인시던트 입력

팁: 한 줄이 타임라인 한 이벤트가 됩니다. 작성자가 다르면 회고 시 책임/협업 흐름을 더 쉽게 추적할 수 있습니다.

생성 결과

아직 생성된 결과가 없습니다.
